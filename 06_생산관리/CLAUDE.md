# 06_생산관리 도메인 규칙

## 목적

설비, 품질, 통합 이슈, 교육/양식 문서를 관리하고, 특히 보전이력과 통계용 템플릿의 기준을 유지한다.

## 읽기 순서

1. `README.md`
2. `STATUS.md`
3. `일일 보전이력.xlsx`
4. `일일 보전이력_통계용 템플릿.xlsx`

## 작업 원칙

- 통계에 쓰일 문서는 코드형 컬럼과 서술형 컬럼을 분리한다
- 설비/품질/자재 분류는 한 원장에서 관리하고, 시트는 보기용으로 분리한다
- 템플릿 개편 시 월간 집계와 관리대장 사용성을 같이 검토한다
- 전역 우선순위와 인수인계는 `90_공통기준/업무관리/TASKS.md`, `HANDOFF.md`를 따른다
