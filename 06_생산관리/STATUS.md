# 06_생산관리 진행 현황

최종 업데이트: 2026-04-11

## 현재 상태

- 설비, 품질, 통합 이슈, 기타 양식이 한 도메인에 공존한다
- `일일 보전이력.xlsx`와 `일일 보전이력_통계용 템플릿.xlsx`를 함께 관리한다
- 설비/품질/자재 분류와 통계용 입력 표준은 도메인 공통 기준으로 유지해야 한다

## 현재 기준 자산

- `통합/`
- `설비/`
- `품질/`
- `기타/`
- `일일 보전이력.xlsx`
- `일일 보전이력_통계용 템플릿.xlsx`

## 다음 작업 후보

- 일일 보전이력 원본과 통계용 템플릿의 컬럼 대응표 작성
- 설비/품질/자재 분류 기준을 코드 관리 시트와 연결
- 월간 집계와 보고서 양식 고정

## 주의사항

- 입력 원장과 보고용 시트를 혼합 운영하지 않는다
- 자유서술 컬럼과 통계용 코드 컬럼을 분리 유지한다
- 전역 작업 상태는 `90_공통기준/업무관리/TASKS.md`를 따른다
