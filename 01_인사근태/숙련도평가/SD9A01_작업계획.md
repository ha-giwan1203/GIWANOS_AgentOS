## 0. 선행 조건 (Phase 0) — 작업표준서 통합

**⚠️ 중요: 작업표준서가 2개 버전 존재 → 통합본 먼저 작성**

| 파일 | 날짜 | Rev | 경로 |
|-----|-----|-----|------|
| 구버전 | 180706 | Rev.16 | `SD9A01_표준문서/SD9A01 작업표준서(180706) Rev.16.xls` |
| 신버전 | 260101 | Rev.16 | `SD9A01_표준문서/SD9A01 작업표준서(260101) Rev.16.xls` |

**처리 방안 (다음 세션에서 사용자 확인):**
- **A안**: 최신본(260101) 단독 사용, 구버전 아카이브로 이동
- **B안**: 양 파일 시트별 비교 → 차이점 추출 → 통합본 작성 (`SD9A01 작업표준서 통합_Rev16.xlsx`)
- **C안**: 사용자가 직접 통합 후 1개 파일만 지정

**권장**: B안으로 자동 비교 후 사용자 검토 (Python으로 시트명/자주검사 항목 diff)

## 2. 작업 단계 (세션108 완료 — 2026-04-25)

### Phase 1 — 작업표준서 통합 ✅
- 입력: `SD9A01 작업표준서(180706) Rev.16.xls` (17시트) + `(260101) Rev.16.xls` (13시트)
- 처리: Excel COM(pywin32)으로 .xls→.xlsx 변환 후 시트 복사 (260101 베이스 + 180706 단독 시트 4개)
- 산출: **`SD9A01 작업표준서 통합_Rev16.xlsx`** (17시트, 29 MB)
  - 시트 순서: 목록 / D9 PT 개정이력 / 20 / 21 / 30 / 40 / 43 / 50 / 60 / 70 / 80 / 81 / 82 / 83 / 90 / 100 / 카메라
- 스크립트: [merge_sd9a01_workstandard.py](생성스크립트/merge_sd9a01_workstandard.py)

### Phase 2 — PROCESSES 11개 데이터 자동 추출 ✅
- 통합본 시트별 "자주검사 항목" 영역(M/N/Q/W 열) 자동 추출 → std 5개
- 관리계획서 공정번호별 행 추출 → ctrl 4개
- 부족 시 GENERIC_STD/CTRL 보강 (안전·이종확인 항목)
- SD9A01_001(PACKAGE 바코드)은 시트 20에서 "바코드" 키워드 항목만 필터, SD9A01_002는 그 외만 (`std_filter='package'`/`'main'`)
- SD9A01_003(신규)은 시트 21 자주검사 단독 사용 + 관리계획서 누락 → GENERIC_CTRL 4개
- 스크립트 내장: [create_sd9a01_v1.py](생성스크립트/create_sd9a01_v1.py) `build_processes()`

### Phase 3 — 공정별 양식 11개 생성 ✅
- SP3M3 샘플 복사 후 PROC1 컬럼(N5/S5/E13~17/E18~21/Q/U/Y) 갱신
- 출력: **`SD9A01_공정별 평가서/SD9A01_공정{001~011} 숙련도 평가서.xlsx`** × 11
- 셀 좌표: SP3M3 v4와 동일 (37r × 95c, 1시트, 67~69KB/파일)
- 스크립트: [create_sd9a01_v1.py](생성스크립트/create_sd9a01_v1.py)

### Phase 4 — 개인별 23명 파일 생성 ✅
- 입력: SD9M01 주간 12명 + 야간 11명
- 마스터 임시 파일에 공정 11개 시트(`공정001~공정011`) 사전 채움
- 작업자별: 수행공정 시트만 유지 + 주공정 맨 앞 + Z3 라벨 + AC열 점수 분배(총점 보존, 0점 없음, 5점 상한, hash 기반 seed로 재실행 동일 결과)
- 출력: **`SD9A01_개인별 평가서/SD9A01 주간|야간/{이름} 숙련도평가서.xlsx`** × 23
- 스크립트: [create_sd9a01_personal.py](생성스크립트/create_sd9a01_personal.py)

### Phase 5 — 검증 ✅
- 파일 수: 공정별 11 + 개인별 23 = **34개** ✓
- 박태순(주공정 003 신규, 10시트) 검증: AC합계 92점 = 원본 일치 / Z3 라벨 정확 / min=4 max=5 (0점 없음)
- 모든 23명 첫 시트 Z3 = '주공정', 나머지 = '전환공정' ✓
- 모든 공정별 양식 37r × 95c, 1시트 ✓
