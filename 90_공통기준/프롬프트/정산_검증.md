# 정산 검증 프롬프트 템플릿

## 프롬프트 본문

```
{정산파일경로} 파일을 검증 스크립트로 검증해줘.

검증 스크립트: 05_생산실적\조립비정산/03_정산자동화/검증_정산결과.py
Python: C:\Users\User\AppData\Local\Programs\Python\Python312\python.exe

검증 후 결과를 아래 형식으로 보고해줘:
- 항목별 PASS/FAIL/INFO 판정
- FAIL 항목: 원인 및 대응 방안
- 종합 판정 요약
```

---

## 사용 시점

- 정산 파일 생성/수정 후 내부 정합성 확인이 필요할 때
- 특정 수치를 수동 수정한 뒤 영향 범위 점검 시
- 월 마감 전 최종 확인 단계

## 필수 입력

| 변수 | 설명 | 예시 |
|------|------|------|
| `{정산파일경로}` | 검증할 정산 결과 엑셀 절대경로 | `05_생산실적\조립비정산/정산결과_02월_수정5.xlsx` |

## 기대 출력

검증 스크립트(`검증_정산결과.py`) 7개 항목 결과 보고:

| # | 항목 | 판정 기준 |
|---|------|---------|
| 1 | DB 정산금액 vs 00_정산집계 | 일치 여부 |
| 2 | 라인합계행 vs 00_정산집계 | 전 시트 일치 여부 |
| 3 | SD9A01/WABAS01/SP3M3 차이 추적 | INFO (원인 분석) |
| 4 | SD9A01 단가기준판정 규칙 | 단가≤500→야간가산, >500→기본 |
| 5 | WABAS01 단가=0 품번 | INFO (건수 및 실적 유무) |
| 6 | Usage=2 수량 2배 환산 | 적용 품번 단가 정합 |
| 7 | 합계행 비교컬럼 오염 | 오염 없음 여부 |

종합: PASS N / FAIL N / INFO N

## 참고

- FAIL 시 STATUS.md 미해결 이슈 섹션에 기록
- 최종 확정 후 DB 정산금액 시트 갱신 필요
