# YouTube 영상 분석 프롬프트

> 스킬: `youtube-analysis`
> 스크립트: `90_공통기준/스킬/youtube-analysis/youtube_transcript.py`
> 용도: URL → 자막 자동 추출 → 분석 → 실무 적용안 출력

---

## 기본 분석

```
이 영상 분석해줘: <YouTube_URL>
```

## Claude 사용환경 개선 목적 분석

```
이 영상 보고 내 Claude Code 설정/스킬/워크플로우에 적용할 수 있는 것들 정리해줘: <YouTube_URL>
```

## 특정 주제 추출

```
이 영상에서 MCP 관련 내용만 뽑아서 정리해줘: <YouTube_URL>
이 영상에서 프롬프트 작성 팁만 뽑아줘: <YouTube_URL>
```

## 타임스탬프 포함 분석

```
이 영상 챕터별로 타임스탬프 포함해서 요약해줘: <YouTube_URL>
```

## 비교 분석 (영상 2개)

```
아래 두 영상을 비교 분석해줘:
영상1: <URL1>
영상2: <URL2>
공통점과 차이점, 각 영상에서 적용할 것들 정리해줘.
```

---

## 출력 포맷 (기본)

| 섹션 | 내용 |
|------|------|
| 요약 | 3~5줄 핵심 |
| 핵심 내용 표 | 주제 / 내용 / 실무 적용 여부 |
| 실무 적용안 | 바로 쓸 수 있는 항목 목록 |
| 다음 액션 | 후속 작업 |

---

## 오류 시 대응

| 상황 | 대응 |
|------|------|
| 자막 없음 | 영상 설명란 URL 요청 또는 수동 복붙 |
| 자막 비활성화 | 동일 |
| 영어 영상 | 영어 자막 추출 후 한국어 번역 분석 |
