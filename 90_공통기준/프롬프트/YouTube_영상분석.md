# YouTube 영상 분석 프롬프트

> 스킬: `youtube-analysis`
> 스크립트: `90_공통기준/스킬/youtube-analysis/youtube_transcript.py`
> GPT 공동작업 합의: 2026-04-01

---

## 자동 모드 (가장 많이 쓰는 방식)

```
영상분석
```

이것만 입력하면 전자동 실행:
1. 현재 시스템 상태 진단 + 중복 검색 방지
2. 실행 가능성 점검 (불가 시 분석 전용 모드 강등)
3. 개선 가설 수립
4. YouTube 검색 (2~5개 영상)
5. 자막 추출 + 분석
6. 공식 문서 교차검증 + A/B/C 판정
7. GPT 토론 (A/B 등급만)
8. A등급 즉시 구현 / B등급 plan.md 생성
9. 상태 문서 갱신 + 결과 보고

---

## 수동 모드 (특정 영상 분석)

```
영상분석: https://youtu.be/xxxxx
```

---

## 변형

```
# 특정 주제 집중
영상분석 hooks 중심으로

# 타임스탬프 포함
영상분석: https://youtu.be/xxxxx (타임스탬프 포함)

# 비교 분석
아래 두 영상 비교 분석:
영상1: <URL1>
영상2: <URL2>
```

---

## A/B/C 판정 기준

| 등급 | 조건 | 후속 |
|------|------|------|
| A | 작고 명확 + 공식 문서 확인 + 되돌리기 쉬움 | 즉시 구현 |
| B | 복수 파일/도메인 + 공식 문서 확인 + plan 필요 | plan.md 생성 |
| C | 환경 미스매치 / 이미 보유 / 증거 불충분 | 참고만 저장 |

## 참고

- 자동 모드에서 중간 승인 불요 (끝까지 자동 진행)
- 분석 관점 9개가 스킬에 기본값으로 설정됨 (매번 설명 불요)
- 영상은 아이디어 소스, 최종 기준은 공식 문서 + 현재 저장소
- GPT 토론 연동은 토론모드 규칙을 따름
