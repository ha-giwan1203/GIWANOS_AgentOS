## 4. Notion DB 확장안

기존 "영상분석 이력" DB에 필드 추가:

| 필드 | 타입 | 값 |
|------|------|-----|
| source_type | select | video / article / docs / repo / talk |
| depth | select | beginner / intermediate / advanced |
| domain | multi_select | Claude Code / evals / manufacturing ERP / harness / operations |
| actionability | select | immediate / needs_design / reference_only |
| applied | checkbox | 적용 여부 |
| evidence_link | url | 적용 시 커밋/문서 링크 |

> 기존 필드(video_id, 채널, 등급 요약, 분석 방식 등)는 유지. source_type=video일 때만 기존 필드 활용.
