# 엑셀 시트/헤더 표준안 v1.0

작성일: 2026-04-09
상태: DRAFT
적용 대상: 관리대장, 템플릿, 분석용 워크북

## 1. 목적

엑셀 파일은 구조가 흔들리면 자동화가 바로 깨진다.
이 문서는 `시트명`, `헤더명`, `컬럼 순서`, `시각 규칙`을 표준화해서
입력, 집계, 자동화, 보고가 모두 같은 기준으로 움직이도록 만든다.

## 2. 표준 시트 구성

기본 구성은 아래 7개 시트를 기준으로 한다.

| 시트명 | 용도 | 비고 |
|--------|------|------|
| 원장 | 전체 사건/입력 데이터 | 기준 시트 |
| 설비이슈 | 설비 필터 시트 | 원장 기반 |
| 품질이슈 | 품질 필터 시트 | 원장 기반 |
| 자재이슈 | 자재 필터 시트 | 원장 기반 |
| 월간집계 | 피벗/집계/차트 | 보고용 |
| 코드관리 | 드롭다운 기준값 | 수정 권한 제한 권장 |
| 가이드 | 작성 예시와 입력 규칙 | 사용자 안내용 |

## 3. 원장 표준 헤더

### 3.1 필수 컬럼

| 순서 | 헤더 | 설명 |

| 1 | 발생일자 | 사건 발생일 |

| 4 | 라인명 | 사람이 보는 라인명 |
| 5 | 공정명 | 공정 또는 작업 위치 |
| 6 | 대분류 | 설비/품질/자재 |
| 7 | 중분류 | 관리용 중분류 |
| 8 | 소분류 | 세부 유형 |
| 9 | 내용요약 | 한 줄 요약 |
| 10 | 원인요약 | 원인 요약 |
| 11 | 조치내용 | 조치 요약 |
| 12 | 담당자 | 작업 또는 등록 담당 |
| 13 | 정지시간(분) | 숫자형 |
| 14 | 금액 | 숫자형 |

## 4. 시트별 헤더 기준

### 4.1 설비이슈 / 품질이슈 / 자재이슈

원장과 동일 헤더를 유지한다.
필터링 또는 보기용 시트라도 헤더명을 바꾸지 않는다.

### 4.2 월간집계

상단 요약:

- 전체 건수
- 설비 건수
- 품질 건수
- 자재 건수
- 총 정지시간
- 총 금액

하단 표:

- 월별 집계
- 중분류 상위 10개
- 공정별 집계
- 금액 상위 건

### 4.3 코드관리

권장 표:

| 항목 | 코드/값 | 사용 여부 | 비고 |
|------|---------|-----------|------|

관리 항목 예시:

- 대분류
- 중분류
- 소분류
- 라인명
- 공정명
- 담당자
- 검토상태

### 4.4 가이드

권장 구성:

1. 작성 목적
2. 한 행 = 한 사건 원칙
3. 헤더별 입력 예시
4. 금지 예시
5. 집계 기준

## 5. 시각 규칙

- 헤더 행은 굵게
- 고정틀 `A2`
- 자동 필터 적용
- 열 너비 자동 조정
- 숫자 컬럼은 오른쪽 정렬
- 날짜 컬럼은 `YYYY-MM-DD`
- 금액 컬럼은 천 단위 구분

## 6. 명명 규칙

### 6.1 시트명

한글 기준 시트명을 우선 사용한다.

허용 기본값:

- 원장
- 설비이슈
- 품질이슈
- 자재이슈
- 월간집계
- 코드관리
- 가이드

### 6.2 파일명

- 템플릿: `주제_template.xlsx`
- 관리형: `주제_management_workbook.xlsx`
- 보고형: `YYYY-MM-DD_주제_report.xlsx`

## 7. 자동화 고려사항

- 헤더명은 띄어쓰기까지 고정한다.
- 시트명 변경 전에는 코드 의존 여부를 확인한다.
- 계산 시트보다 원장 시트를 자동화 기준으로 삼는다.
- 보기 시트는 새로 만들어도 원장 구조는 유지한다.
