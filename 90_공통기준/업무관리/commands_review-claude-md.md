# CLAUDE.md 리뷰

프로젝트의 모든 CLAUDE.md 파일을 검토하고 문제를 보고하세요.

## 검사 대상
- 루트 `CLAUDE.md`
- `05_생산실적/조립비정산/CLAUDE.md`
- `10_라인배치/CLAUDE.md`

## 검사 항목

### 1. 규칙 혼재 검사
- 루트 CLAUDE.md에 특정 도메인 전용 규칙이 섞여있지 않은지
  - 예: 조립비 정산 전용 계산 규칙이 루트에 있으면 FAIL
  - 예: 라인배치 ERP URL이 루트에 있으면 FAIL
- 도메인 CLAUDE.md에 프로젝트 공통 규칙이 중복 선언되어 있지 않은지

### 2. 상태 정보 누수 검사
- CLAUDE.md에 날짜/버전/진행상태 같은 휘발성 정보가 포함되어 있지 않은지
  - 규칙 문서에는 "무엇을 해야 하는지"만 있어야 함
  - "현재 어디까지 했는지"는 STATUS.md/TASKS.md에 있어야 함
- 예외: 스킬 사용 기준표의 스킬 목록은 규칙이므로 OK

### 3. Plan-First 준수 확인
- 루트 CLAUDE.md에 Plan-First 워크플로우 3규칙이 있는지
  - research.md 선행
  - plan.md 승인 전 구현 금지
  - plan.md 인라인 주석 수정
- 도메인 CLAUDE.md에서 Plan-First를 무시하는 규칙이 없는지

### 4. 공동작업 원칙 확인
- 루트 CLAUDE.md에 공동작업 운영 원칙 5항목이 있는지
- 공동작업 응답 표 금지 규칙이 출력 형식에 있는지

### 5. 하네스 검증 원칙 확인
- 루트 CLAUDE.md에 하네스 검증 원칙 4항목이 있는지
- skill-creator 전용 문구가 루트에 섞여있지 않은지

## 출력 형식
각 검사 항목별 PASS / FAIL / WARNING.
FAIL이 있으면 해당 파일, 줄 번호, 문제 내용, 권장 수정 방향 표시.
