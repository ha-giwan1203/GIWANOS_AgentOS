## 🔲 테스트 시나리오 (v1.0 검증용)

### 시나리오 1 — 기본 1턴 왕복
```
주제: "AI가 제조업 현장 관리자 업무를 대체할 수 있는가"
Claude 역할: 반대측 (현장 리스크 중심 반박)
GPT 역할: 찬성측 (효율화 주장)
예상 흐름:
  1. Claude → GPT에 찬성 주장 요청 입력
  2. 완료 감지 (2중 신호) → assistant 블록 추출
  3. Claude가 반박 생성 → GPT에 입력
  4. 로그 저장
검증 항목:
  - 완료 감지 정확도 (잘린 텍스트 없는지)
  - assistant 블록 식별 정확도
  - 로그 파일 생성 여부
```

### 시나리오 2 — 3턴 연속 토론
```
주제: "Claude와 ChatGPT 중 어느 쪽이 실무 자동화에 더 적합한가"
라운드: 3턴
종료 조건: 3라운드 완료 또는 "합의" 감지
검증 항목:
  - 라운드 간 컨텍스트 유지 여부
  - 반복 주장 감지 로직 동작 여부
  - 전체 로그 무결성
```

### 시나리오 3 — 오류 복구
```
상황: GPT 응답 중간에 read_page 호출 (의도적 타이밍 오류)
예상 동작:
  - 완료 감지 실패 → 5초 대기 후 재시도
  - 3회 실패 시 토론 중단 + 에러 로그 저장
검증 항목:
  - 오류 복구 로직 동작 여부
  - errors/ 폴더 로그 생성 여부
```
