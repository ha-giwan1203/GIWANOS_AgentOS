# 3턴 테스트 시나리오

## 테스트 조건
- 주제: "AI가 인간의 일자리를 대체할 것인가"
- Claude 관점: 현실검증자
- 턴 수: 3

## 기대 흐름
Turn 1: Claude가 GPT에 주제 질문 → GPT 응답
Turn 2: Claude가 GPT 응답에 반박 → GPT 재응답
Turn 3: Claude가 재응답에 추가 반박 → GPT 최종 응답
