## 핵심 동작 관찰

1. **그리드 행 = 검사항목 단위 누적**: 한 검사항목당 그리드 1행. 검사항목 변경 시 해당 행만 표시 (다른 검사항목 행은 숨김).
2. **재입력 정정 가능**: 동일 (제품/공정/검사항목)에 재입력 → 재저장하면 덮어쓰기 (사용자 답변 + 실측 확인). R5 롤백 경로 = 자동 재실행이 곧 롤백.
3. **검사항목 수 N≥1**: [200]은 5개. 한 공정에 여러 검사항목 정상.
4. **자동 작업자 매핑**: 일부 공정([91] 등) 진입 시 우측에 작업자 사진/이름 자동 노출. 작업자 인증 미선행이어도 작동 (단독 처리 가능).
5. **응답 딜레이**: 공정 변경 직후 SmartMES 화면 갱신 1~3초 지연. 자동화 시 매 변경 후 `wait(2)` 필수. "잡셋업 검사 결과를 불러올 수 없습니다" 빨간 메시지가 가끔 뜨지만 그리드는 정상 노출됨.
