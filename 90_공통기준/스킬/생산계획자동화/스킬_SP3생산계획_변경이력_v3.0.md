# SP3 생산계획 변경이력 v3.0

## v3.0
### 1. 자동계획 반영
- `APPLY_TO_PLAN` 기준으로 자동 생산계획 생성 구조 정리
- PLAN_RESULT_FIXED → Sheet4 반영 흐름 확정

### 2. 수동서열 기능
- V/W 수동서열 기능 구현 완료
- `APPLY_TO_PLAN` 실행 시 수동서열 자동 반영 포함
- `APPLY_MANUAL_SEQ` 별도 실행 가능
- V/W는 유지형 입력이 아니라 1회성 명령값으로 변경
- 반영 후 V/W 자동 초기화 적용

### 3. Sheet4 버튼 이동 기능
- `SetupMoveButtons` 기반 이동 버튼 운영 추가
- `MoveSelectedUp / MoveSelectedDown` 브릿지 구조 확정
- `ExecMoveUp / ExecMoveDown` 실제 이동 로직 반영

### 4. 다중선택 이동 개선
- 다중선택 교차이동 허용
- 주야간 블록 이동 시 배열 임시저장 후 삽입 방식으로 변경
- 빈 행 포함 블록 이동 차단
- 연속 블록만 허용

### 5. 선택 유지 개선
- 이동 후 선택이 단일 셀로 풀리던 문제 수정
- `ReselectBlock` 기준으로 이동 후 블록 재선택 유지

### 6. 운영기준 정정
- Sheet4 직접 잘라내기/붙여넣기 금지 유지
- 위치 보정 수단을 `V/W` 또는 버튼 이동으로 확정
- 수식열 직접 이동/수정 금지 원칙 재확인

### 7. 문서 구조 개편
- 기존 단일 문서에 섞여 있던 스킬 / 운영절차 / 변경이력 분리
- 스킬 문서는 고정 규칙만 유지
- 운영절차는 실행 순서만 분리
- 변경이력은 구현 변경점만 기록
