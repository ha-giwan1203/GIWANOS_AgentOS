# 임률단가 관리 스킬

## 목적
공정별 임률단가의 연도별 갱신과 원청/업체/대원테크 3계층 단가 관리를 자동화한다.

## 대상 파일

| 파일 | 경로 | 역할 |
|------|------|------|
| 연도별 임율 기준 | `02_급여단가/임률단가/01_원청기준/임률단가_연도별임율기준_22-25년.xlsx` | 연도별 기준 원본 |
| 원청 기준 최종본 | `02_급여단가/임률단가/01_원청기준/임률단가_원청기준_최종본_v6.xlsx` | 원청 단가 기준 |
| 업체 기준 최종본 | `02_급여단가/임률단가/02_업체기준/임률단가_업체기준_최종본_v10.xlsx` | 업체별 단가 |
| 대원테크 독립계산 | `02_급여단가/임률단가/03_대원테크/임률단가_대원테크_독립계산.xlsx` | 대원테크 전용 단가 (조립비 정산 연동) |

## 운영 절차

### 분기별 갱신
1. 원청 기준 단가 변경 여부 확인
2. 변경 시 원청 기준 최종본 갱신
3. 업체 기준에 원청 변경분 반영
4. 대원테크 독립계산 파일 연동 갱신
5. 조립비 정산 기준정보와 일치 여부 검증

### 연도별 갱신 (연초)
1. 전년도 기준 복사
2. 신규 연도 임률 적용
3. 전 업체 단가 일괄 갱신

## 연관 스킬
- assembly-cost-settlement: 대원테크 단가를 조립비 정산 기준정보로 사용
- production-report: 임률단가를 생산관리 마스터리스트에 참조

## 금지사항
- 원청 기준 파일을 업체 기준으로 역방향 수정 금지
- 검증 없이 단가 일괄 변경 금지
- 대원테크 독립계산 파일의 수식 구조 임의 변경 금지
