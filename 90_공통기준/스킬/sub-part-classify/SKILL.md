# sub-part-classify — 라인별 서브품 운영 LIST 자동 채움

## 목적

`서브품 운영 대상 LIST.xlsx` (사용자 데스크탑) 각 라인 시트의 적용차종·적용품번을 라인 기준정보에서 자동 추출·입력한다.

## 라인 지원

| 라인 | LIST 시트 | 기준정보 파일 | 기준정보 시트 | 매핑 룰 |
|------|----------|------------|------------|---------|
| SP3M3 | `대원테크_SP3M3` | `04_생산계획/SP3M3_생산지시서_(YY.MM.DD).xlsm` | `SP3 LINE 기준 정보` | 비고 키워드 → C9(홀더CLR 사양) 정확 일치 |
| SD9A01 | `대원테크_SD9A01` | `\\210.216.217.180\zz-group\15. SP3 메인 CAPA점검\SP3M3\SD9M01 라인 계획표.xlsm` | `기준정보` | M열(디링 ASSY) + O열(앵카 ASSY) unique 추출 |

## 공통 룰

### 차종 추출 (옵션2 — 첫 단어 + 괄호 제거)
- 입력: 기준정보 차종 셀 (예: `JX0\n(GV80)`, `OVK (90도)\n(EV5)`, `MX5A (MDS)\n싼타페`)
- 처리: 줄바꿈 앞 → 공백 split[0] → 괄호 이후 제거 → unique
- 결과: `JX0`, `OVK`, `MX5A`

### 빈 값/오류 필터
- 모듈품번(C5) 또는 ASSY 품번 셀이 `None`, 빈 문자열, `'X'`, `#N/A`/`#REF!`/`#VALUE!`(수식 오류)이면 해당 행 스킵.

### LIST 입력 형식 (블록 5행)
| 행 | C(구분) | D(값) |
|----|---------|------|
| +0 | 품번 | 모듈품번/ASSY 품번 |
| +1 | 품명 | (라인별 지정) |
| +2 | 적입수량 | (라인별 지정) |
| +3 | 적용차종 | 콤마 구분 unique (공백 없음) |
| +4 | 적용품번 | 콤마 구분 unique (공백 없음) |

NO 셀(B 컬럼)은 5행 병합. 서식은 직전 블록 템플릿 복제.

## SP3M3 라인 매핑 사전 (사용자 비고 → C9)

| 사용자 비고 | C9 정확 일치 키 |
|----------|--------------|
| CLR흰색 | `CLR 흰색` |
| CLR 노랑 | `CLR 노랑` |
| NLR 흰색 | `NLR 흰색` |
| NLR 노랑 | `NLR 노랑` |
| P/T CLR흰색 | `P/T CLR흰색` |
| P/T CLR노랑 | `P/T CLR노랑` |
| P/T NLR흰색 | `P/T NLR흰색` |
| P/T NLR노랑 | `P/T NLR노랑` |
| MDS CLR노랑(MDS) | `P/T CLR노랑\n(MDS)` |
| MDS 커버 | (기준정보 부재 — 같은 그룹 SUB 동일값 적용) |

**중요 — 회귀 방지**:
- C9 단독 정확 일치. 모델(C16)·공급라인(C19) 필터 적용 금지. (1차 작업 시 SP2M02·P/TRAY 필터로 좁혀서 매칭 누락 회귀 발생)
- 사용자 비고 "빨강"은 기준정보 "흰색"과 동일 매핑 (사용자 표기 정정 후 일치 확인됨).

## SD9A01 라인 — 디링/엥카 ASSY 처리

- 디링 ASSY: M열 (C13) — 품명 `디링 ASSY`, 적입수량 `60`
- 앵카(엥카) ASSY: O열 (C15) — 품명 `엥카 ASSY`, 적입수량 `120`
- 각 ASSY 품번 unique → 그 코드가 등장한 행들의 차종(C3)·품번(C2/C6) 집계.
- 디링 먼저, 엥카 뒤. 각 그룹 알파벳 정렬.

## 사용법

```bash
python run.py --line SP3M3
python run.py --line SD9A01
```

옵션:
- `--dry-run`: LIST 파일 저장하지 않고 결과만 출력.
- `--backup-only`: 백업만 생성하고 종료.

실행 전 자동 백업: `서브품 운영 대상 LIST.xlsx.bak_YYYYMMDD_HHMMSS`

## 검증 (필수)

1. 실행 후 LIST.xlsx Excel에서 열어 시각 확인 (memory `feedback_verify_with_screenshot.md`).
2. 1번 샘플 행이 손상되지 않았는지 확인.
3. SP3M3 SSP3-0021 샘플 = 사용자 수작업 10개 품번 일치 (`CLR 흰색` 단독 매핑).
4. SD9A01 디링 35개 + 엥카 25개 = 합 60 ASSY (2026-05-20 기준).
