# sp3-production-plan — MANUAL

> 상세 절차/규칙/실패조건/되돌리기. SKILL.md는 호출 트리거 + 80줄 요약만.

## 대상 파일
SP3 생산지시서 최종본 (엑셀, 10개 시트)

| 시트 | 역할 |
|------|------|
| SP3 LINE 기준 정보 | SUB 기준정보 마스터 |
| OUTER 생산계획 | ERP 계획 원본 붙여넣기 |
| 재고조사 | 현재고 입력 + D+1/D+2 필요 계산 |
| 생산계획 | 최종 생산지시 출력 |
| 계획 확인 | D+1/D+2 지시량 검증 |
| PLAN_RESULT_FIXED | 매크로 중간 결과 |
| PLAN_BACKUP_FIXED | 생산계획 백업 |

## 일일 운영 순서
1. Sheet2에 ERP 계획 붙여넣기
2. Sheet3에 D+1/D+2 현재고 입력
3. 매크로 `APPLY_TO_PLAN` 실행
4. Sheet4 야간/주간 정상 반영 확인
5. 필요 시 수동서열(V/W열) 입력 후 `APPLY_MANUAL_SEQ`
6. 최종 Sheet4 출력 확인

## 핵심 규칙

### 생산 구간
- D+1 = 야간(행 3~100), D+2 = 주간(행 104~203)
- 야간 3,900EA 초과 시 → 초과 첫 품번까지 야간 포함, 이후 주간 이월

### 수식 보호
- 매크로가 쓰는 열: B/E/G/T만
- 수식열(C,D,F,H~S,K,M): 절대 직접 덮어쓰기 금지

### 자동 정렬 (13단계)
SUB 823 마지막 → 8PI/10PI → 토션순 → P/TRAY 위치 → 각도 → 모델 → 차종 → 하우징 → 홀더 → 센스 → 특이사항 → SUB 품번 → 동률 원본 유지

### 수동서열
- V열: 야간/주간, W열: 목표순번
- 1회성 명령값 (반영 후 자동 초기화)
- `APPLY_TO_PLAN` 또는 `APPLY_MANUAL_SEQ`로 반영

## 매크로 체계
| 매크로 | 역할 |
|--------|------|
| APPLY_TO_PLAN | 자동계획 생성 + 반영 + 수동서열 |
| PREVIEW_BUILD | 결과 미리보기만 |
| APPLY_MANUAL_SEQ | 수동서열만 재반영 |
| RESTORE_PLAN | 백업 복원 |

## 실패 조건
| 조건 | 판정 |
|------|------|
| 생산지시서 파일 열기 실패 | FAIL |
| ERP 계획 Sheet2 데이터 0건 | FAIL |
| 매크로 APPLY_TO_PLAN 실행 에러 | FAIL |
| 야간/주간 구간 모두 데이터 0건 | FAIL |
| 수식열 값 파괴 감지 | FAIL |

## 중단 기준
1. 수식열 변경 감지 — 구조 파괴 위험
2. Sheet4 직접 편집 시도
3. 매크로 실행 후 V/W열 미초기화
4. 823 마지막 정렬 깨짐

## 검증 항목
- B/E/G/T열만 입력 확인 (수식열 미침범)
- K/M 수식열 보존
- 야간/주간 순번 연속 재부여 정상
- V/W 자동 초기화 완료
- 823 마지막 정렬 유지

## 되돌리기
| 범위 | 방법 |
|------|------|
| 계획 전체 | RESTORE_PLAN 매크로 (PLAN_BACKUP_FIXED 시트에서 복원) |
| ERP 데이터 | Sheet2 ERP 원본 재붙여넣기 |
| 재고 데이터 | Sheet3 재입력 후 매크로 재실행 |

## 금지
- Sheet4 직접 잘라내기/붙여넣기
- 수식열 직접 수정
- ERP 원본을 Sheet2 외 시트에 덮어쓰기
