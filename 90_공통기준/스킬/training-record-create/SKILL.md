# 교육 기록부 작성 스킬

## 목적
- 대원테크 표준 양식(`대원테크_부적합교육_{라인}_{주제}_{YYYYMMDD}.xlsx`)으로 라인 교육 기록부 1건을 즉시 생성한다.
- 기준 양식: `06_생산관리/품질/교육기록부/대원테크_부적합교육_MX5_바코드이종_20260428.xlsx` (시트 `교육기록부` 단일, 병합셀 187개)
- 메타·본문 채움 + 평가/명단/결재는 사용자 사후 기입용 빈 양식.

## 트리거
- "{라인} 교육 기록부 만들어"
- "{라인} 부적합 교육 자료 작성"
- "라인 내 8대 유의사항", "재발방지 교육" 등 교육용 자료 + 라인 지정 발화
- 사용자가 이미지·텍스트로 교육 본문 자료 제공 + "교육 기록부" 키워드

## 입력
- **라인** (필수): SD9A01 / SP3M3 / MX5 / QX1 / JX / MQ4 등
- **제목** (필수): 교육 주제 (예: "라인 내 8대 유의사항", "MGG 역방향 조립 부적합 교육")
- **본문** (필수): 교육 내용 텍스트 (이미지에서 추출하거나 사용자 제공)
- **일자** (옵션, default 오늘 KST)
- **교육담당** (옵션, default 하지완 과장)
- **교육시간** (옵션, default 30분)
- **사내·외** (옵션, default 사내)

## 출력
- 신규 파일: `06_생산관리/품질/교육기록부/대원테크_부적합교육_{라인}_{주제슬러그}_{YYYYMMDD}.xlsx`
- 채워진 셀: G6 일자 / V6 사내외 / G7 담당 / V7 시간 / G8 대상자 / V8 인원 / G9 제목 / G10 본문 / E44 키워드 / T45 교육자
- 빈 양식 유지: 결재(W3·AC3), 평가(B38), 명단(row 49~65 D·J·M / S·Y·AB)

## 실행 절차
1. 기준 양식 복사
   - `cp 06_생산관리/품질/교육기록부/대원테크_부적합교육_MX5_바코드이종_20260428.xlsx <TARGET>`
2. `python run.py --line <LINE> --topic <TOPIC> --body-file <BODY.txt> [--date YYYY-MM-DD] [--instructor <NAME>]`
3. 메타 8개 셀 + 본문 G10(wrap_text=True) + E44/T45 갱신
4. 명단·평가·결재 빈 양식화 (기존 MX5 데이터 삭제)
5. 저장 → openpyxl 재로딩 검증

## 실패 조건
- 기준 양식 파일 부재
- 시트명 `교육기록부` 불일치 (양식 변형 시)
- 병합셀 수 187 ± 2 벗어남 (양식 깨짐)
- 본문 길이 5000자 초과 (한 셀 wrap 가독성 한계)

## 중단 기준
- 동일 파일명 이미 존재 → 사용자 통보 후 덮어쓰기 여부 자체 판단 (default 덮어쓰기 금지, `_v2` 부여)
- 본문 텍스트 인코딩 오류 (cp949 → utf-8 변환 실패)

## 검증 항목
- 신규 파일 size > 15KB
- G6 datetime 정상 / G8 라인명 일치 / G9 제목 일치
- G10에 본문 핵심 키워드 2개 이상 포함 (사용자 제공 키워드 또는 첫째·마지막 항목)
- 시트 `교육기록부` 단일, 병합셀 수 187 유지
- 명단 D49 / J49 빈칸 (사용자 사후 기입 영역)

## 되돌리기 방법
- 신규 파일이므로 단순 삭제로 원복: `rm 대원테크_부적합교육_{라인}_{주제}_{YYYYMMDD}.xlsx`
- 기존 양식 4개(MX5·QX1×2·통합)는 미수정 — 영향 없음
