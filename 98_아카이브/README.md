# 98_아카이브

이 폴더는 완료 자료, 정리 대기 자료, 마이그레이션 증적을 보관하는 구간이다.

## 보관 대상

- 완료 후 직접 수정하지 않을 산출물
- 폴더 정리 과정에서 분리한 이전 버전
- 구조 개편 증적, 마이그레이션 결과, 임시 보류 묶음

## 하위 폴더 규칙

- `마이그레이션_YYYYMMDD/`
- `정리대기_YYYYMMDD/`
- 필요 시 작업명 접미사 추가

## 운영 원칙

1. 활성 원본 파일은 이 폴더로 먼저 옮기지 않는다.
2. 아카이브 이동 시 날짜와 이동 이유가 폴더명이나 메모에 드러나야 한다.
3. 다시 활성화해야 하는 파일은 원래 도메인으로 복귀시키고, 이 폴더를 원본처럼 직접 수정하지 않는다.
