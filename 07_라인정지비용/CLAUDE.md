# 라인정지비 도메인 규칙

## 데이터 원본 — G-ERP 라인보상상세현황

**메뉴 경로**: 삼송 G-ERP → 클레임관리 → 라인보상관리 → **라인보상상세현황**

### 검색조건 (월 정산용 표준)
- 발생일자: 정산 대상월 1일 ~ 말일
- 조립업체: 대원테크 (0109)
- 청구유형 / 발생 라인 / 발생 유형: 전체

### 상단 그리드 (47건 기준 4월 사례)
| 컬럼 | 비고 |
|------|------|
| 청구유형 | "라인정지"·"재작업"·"선별작업"·"라인교체"·"추가CLAIM" |
| 청구번호 | `0109-YYYYMMDD-S###` 패턴 |
| 승인상태·접수상태 | 둘 다 "승인"·"접수" 정상 |
| 귀책구분 | 사내부서 / 납품업체 |
| 귀책업체 | 품질관리팀 / 생산기술팀 / 웰텍스 등 |
| 품번 | RSP3SC*·RSP3PC*·RSP3AC* (SP3M3 모듈품번) |
| 발생 라인·공정그룹 | SP3M3 / G08_SP3M3 (4월 47건 전부) |

### 하단 상세 (라인정지 탭) — 비용 산출 필드
- 비가동 시작일시 / 종료일시
- 휴식시간(분) / 작업인원
- **분당정지비(원)** — 주간·야간 분리
- **정지시간(분)** — 주간·야간 분리

### 산출 공식
```
라인정지비 = (주간 분당정지비 × 주간 정지시간) + (야간 분당정지비 × 야간 정지시간)
```

## 월별 작업 절차

1. G-ERP 라인보상상세현황 → 월 검색 → **엑셀 다운로드**
2. `05_생산실적/조립비정산/{MM+1}월/라인정지_{MM}월_raw.xlsx` 저장
3. 청구번호별 정지비 합산 → 라인별·귀책업체별 집계
4. 조립비 정산 본체(`정산_수식버전_MM월.xlsx`)에 별도 시트 또는 보조 파일로 첨부

## 작업폴더 관행
- 매월 정산 작업폴더 = `05_생산실적/조립비정산/{MM+1}월/` (조립비정산과 동일)
- 라인정지비 raw·집계 산출물은 정산 작업폴더에 같이 두는 것을 default로 한다 (조립비 정산과 함께 보고)
