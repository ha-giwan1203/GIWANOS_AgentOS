# 02_급여단가 도메인 규칙

## 목적

임률단가 산정 파일을 원청 기준, 업체 기준, 대원테크 독립 계산 구조로 관리한다.

## 읽기 순서

1. `README.md`
2. `STATUS.md`
3. `임률단가/` 하위 실파일

## 작업 원칙

- 원본 엑셀은 입력/계산/요약 시트 구조를 먼저 확인한 뒤 수정한다
- 기준 변경 시 원청/업체/대원테크 세 파일군 영향 여부를 같이 본다
- 보고값은 연도, 기준, 적용 단가 출처를 함께 기록한다
- 전역 우선순위와 인수인계는 `90_공통기준/업무관리/TASKS.md`, `HANDOFF.md`를 따른다
