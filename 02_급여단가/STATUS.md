## 현재 상태

- 도메인 진입 문서 정리 완료
- 실작업 원본은 `임률단가/` 하위 엑셀 파일 3계층 구조다
- 자동화 스크립트와 검증 체계는 아직 도메인 문서로 고정되지 않았다

## 현재 기준 파일

- `임률단가/01_원청기준/`

- 전역 작업 상태는 `90_공통기준/업무관리/TASKS.md`를 따른다
