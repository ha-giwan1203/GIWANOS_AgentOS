## 공용품번 박스 최종 규격

### 셀 분할
| 영역 | 행 범위 | 내용 | 폰트 | 색상 |
|------|---------|------|------|------|
| 헤더 | AB1:AJ4 | "공용 품번" | 원본유지 | 배경 하늘색 |
| 1단 | AB5:AJ9 | SP3M3 | 85pt Bold HY헤드라인M | 검정 |
| 2단 | AB10:AJ12 | 공용품번 | 110pt Bold HY헤드라인M | LH빨강/RH파랑 |
| 3단 | AB13:AJ14 | RSP | 80pt Bold HY헤드라인M | 검정+부분색상 |

### RSP 부분색상
- RSP3 = 검정
- 중간영문(SC/SE/PC/HE/AC) = LH빨강(ColorIndex 10) / RH짙은파랑(ColorIndex 12)
- 숫자 = 검정

### LH/RH 판정
- H12 앞5자리: 88810, 89870 = LH / 88820, 89880 = RH

## 케이스별 처리

| 케이스 | 처리 방식 |
|--------|----------|
| 입력 (RSP없음) | C안 3분할, 기준정보 RSP 입력 |
| 수정 (RSP다름) | C안 3분할, 기준정보 RSP로 교체 |
| 유지 (RSP같음) | C안 3분할, 기존 RSP 유지, 라인코드만 변경 |
| 미매칭+RSP있음 | C안 3분할, 기존 RSP 유지, 라인코드만 변경 |
| 미매칭+SUB있음 | C안 3분할, 시트명→공용품번, RSP 비움 |
| 미매칭+empty | C안 3분할, 시트명→공용품번, RSP 비움 |

## 매칭 로직
