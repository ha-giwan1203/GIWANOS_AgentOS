# 초물표 모듈품번 일괄 반영 작업가이드

## 1. 작업 개요

초물표 파일들의 각 시트 상단 공용 품번 박스에 모듈품번(RSP...)을 반영/검증한다.
라인코드 SP3S03 → SP3M3 변경, RSP 입력/수정, 뷰/구분선 통일을 포함.

---

## 2. 파일 위치

| 항목 | 경로 |
|------|------|
| 작업 폴더 | `C:\Users\User\Desktop\업무리스트\03_품번관리\초물관리` |
| 원본 백업 | `_backup\` (xls 원본 49개, 절대 수정 금지) |
| 결과 출력 | `_output\` (복사본만 수정) |
| 기준정보 | `SP3M3_모듈품번_최신.xlsx` (컬럼: 차종/품번/SUB품번/모듈품번) |

---

## 3. 매칭 로직

- SUB품번 기준 사용 금지 (62건 중복 존재)
- **H12 품번 정규화** 기준으로 기준정보 매칭
- 정규화: 대문자 변환 + 공백 제거 + 하이픈 제거
- H12가 멀티라인이면 첫 번째 품번만 사용
- 예시: `88810-CU100` → `88810CU100` → 기준정보 품번 `88810CU100` → RSP3SC0509

---

## 4. AB5 파싱 규칙

- 줄 분리 후 공백 trim + 빈 줄 제거
- 1줄째 = 라인코드, 2줄째 = 공용품번, 3줄째 = RSP (있으면)
- RSP 존재 판정: 3줄째가 "RSP"로 시작하면 있음, 2줄만 있으면 없음

---

## 5. 작업 구조 — 2단 분리

### [1단계: 전체 공통 적용] 모든 시트 — 뷰/구분선만

- Normal View 강제 (ActiveWindow.View = 1)
- DisplayPageBreaks = False
- ResetAllPageBreaks()
- 구분선 통일
- AB5(라인코드/RSP/폰트/색상) 일체 미포함 → 전부 2단계

### [2단계: RSP 값 처리] H12 매칭 후 분기

#### 판별 기준

| 조건 | 판정 |
|------|------|
| AB5에 RSP 없음 + 기준매칭 | **입력** |
| AB5에 RSP 있음 + 기준RSP와 다름 | **수정** |
| AB5에 RSP 있음 + 기준RSP와 같음 | **값 유지** |
| H12 기준정보 미매칭 | **값 변경 없음** |

---

## 6. 케이스별 처리 규칙

### [입력] 원본 2줄 → 3줄 추가

- 기존 AB5 2줄째(공용품번)를 먼저 읽어서 그대로 사용
- C안 셀 분리 적용
- AB5:AJ14 병합 해제 → 3분할 재병합
  - AB5:AJ9 = SP3M3 (Size=85, 검정, HY헤드라인M, Bold)
  - AB10:AJ12 = 기존 AB5 2줄째 원본값 (Size=110, LH빨강/RH파랑, HY헤드라인M, Bold)
  - AB13:AJ14 = 기준정보 RSP (Size=80, 검정+부분색상, HY헤드라인M, Bold)
- 각 영역 가로/세로 가운데 정렬, WrapText=False, ShrinkToFit=False

### [수정] 기존 RSP ≠ 기준 RSP

- C안 셀 분리 통일 적용
- AB5:AJ14 병합 해제 → 3분할 재병합
  - AB5:AJ9 = SP3M3 (Size=85, 검정, HY헤드라인M, Bold)
  - AB10:AJ12 = 기존 AB5 2줄째 원본값 (Size=110, LH빨강/RH파랑, HY헤드라인M, Bold)
  - AB13:AJ14 = 기준정보 RSP (Size=80, 검정+부분색상, HY헤드라인M, Bold)
- 기존 AB5 값을 줄 분리 → 2줄째(공용품번)를 그대로 읽어둠
- 공용품번은 시트명이 아니라 기존 AB5 2줄째 원본값 사용

### [값 유지] 기존 RSP = 기준 RSP

- C안 셀 분리 통일 적용
- AB5:AJ14 병합 해제 → 3분할 재병합
  - AB5:AJ9 = SP3M3 (Size=85, 검정, HY헤드라인M, Bold)
  - AB10:AJ12 = 기존 AB5 2줄째 원본값 (Size=110, LH빨강/RH파랑, HY헤드라인M, Bold)
  - AB13:AJ14 = 기존 RSP (Size=80, 검정+부분색상, HY헤드라인M, Bold)
- 1줄째 라인코드 SP3S03 → SP3M3 변경

### [미매칭+RSP있음] 기준정보 없음 — 기존 RSP 유지

- C안 셀 분리 통일 적용
- AB5:AJ14 병합 해제 → 3분할 재병합
  - AB5:AJ9 = SP3M3 (Size=85, 검정, HY헤드라인M, Bold)
  - AB10:AJ12 = 기존 AB5 2줄째 원본값 (Size=110, LH빨강/RH파랑, HY헤드라인M, Bold)
  - AB13:AJ14 = 기존 RSP (Size=80, 검정+부분색상, HY헤드라인M, Bold)
- 1줄째 라인코드 SP3S03 → SP3M3 변경

### [미매칭+empty] 공용품번 비어있는 시트

- C안 셀 분리 통일 적용
- AB5:AJ9 = SP3M3 (Size=85, 검정)
- AB10:AJ12 = **시트명** (Size=110, LH빨강/RH파랑)
- AB13:AJ14 = 비움 (Size=80, 검정)

---

## 7. RSP 부분색상 (입력/수정 공통)

- RSP3 = 검정
- 중간영문 = **코드별 고유 색상** (LH/RH 무관)
- 뒤 숫자 = 검정

| 코드 | 색상 | BGR값 |
|------|------|-------|
| SC | 빨강 | 0x0000FF |
| SE | 초록 | 0x008000 |
| PC | 보라 | 0x800080 |
| HE | 주황 | 0x0080FF |
| AC | 갈색 | 0x004080 |

---

## 8. 배경색 규칙

| 영역 | 배경색 | 값 |
|------|--------|-----|
| AB1:AJ4 (헤더) | 하늘색 | 0xFFCC99 (ColorIndex 37) |
| AB5:AJ14 (내용) | 흰색 | 0xFFFFFF |

## 9. 공용품번 박스 안쪽 구분선

- AB5:AJ14 InsideHorizontal = xlNone (-4142)
- AB5:AJ14 InsideVertical = xlNone (-4142)
- 3분할 영역 사이 경계선 없음

## 10. LH/RH 판정

- H12 앞5자리: 88810, 89870 = **LH** (빨강, ColorIndex 10)
- H12 앞5자리: 88820, 89880 = **RH** (짙은파랑, ColorIndex 12)

---

## 9. 수정 케이스 자체검증

적용 직후 즉시 확인:
- 1줄째 = SP3M3
- 2줄째 = 기존 공용품번과 동일
- 3줄째 = 새 RSP
- 줄별 서식 복원 성공 여부
- 실패 시 해당 시트 FAIL 처리, 다음 시트 진행

---

## 10. 실행 순서

### Step 1: JW1 재테스트
- _backup에서 재복사
- 1단계: 전체 시트 공통 적용 (뷰/페이지 나누기/구분선)
- 2단계: 901=수정, 902=수정, 911~922=미매칭(라인코드만)
- 육안 확인 후 PASS/FAIL

### Step 2: 전체 일괄 적용 (JW1 PASS 후)
- _backup 49개 파일 순회
- 1단계 + 2단계 적용
- 결과 로그 출력

---

## 11. 검증 항목

1. 전 시트 라인코드 SP3M3
2. 공용품번 원본 유지 (AB5 2줄째 원본 동일)
3. RSP H12 기준 매칭 정확
4. LH빨강 / RH짙은파랑
5. 전 케이스 C안 셀 분리(AB5:AJ9/AB10:AJ12/AB13:AJ14) 통일
6. Normal view (파란 점선/회색 없음)
7. 저장 후 재열기 정상
8. 미매칭/유지 시트 RSP 원본 그대로

---

## 12. 금지사항

- _backup 원본 직접 수정
- SUB품번 기준 매칭
- AB5 공용품번(2줄째)을 시트명이나 변수로 대체
- C안 셀 분리 없이 AB5 값만 재작성
- C안 셀 분리 기준 외의 병합 구조 사용
- 기준정보 확인 없이 임의 입력
- 파일명/시트명/코드체계 임의 변경
