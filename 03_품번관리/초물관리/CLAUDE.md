# SP3M3 초물표 프로젝트

## 목적
초물표 파일의 공용품번 박스에 모듈품번(RSP) 반영, 라인코드 SP3S03→SP3M3 변경, 뷰/구분선 통일.

## 폴더 구조
- `_backup/` — 원본 xls 49개 (절대 수정 금지)
- `_output/` — 최종 결과 49개
- `_scripts/` — 최종 스크립트 (`_apply_all_v3.py`, `_patch_v4.py`)
- `SP3M3_모듈품번_최신.xlsx` — 기준정보 (품번→RSP 매핑)
- `초물표_모듈품번_작업가이드.md` — 상세 작업 규칙

## 작업 전 필수 확인
1. 상위 `업무리스트/CLAUDE.md` 규칙을 먼저 읽는다
2. `초물표_모듈품번_작업가이드.md`를 읽고 현재 기준을 확인한다
3. `_backup/` 원본은 절대 수정하지 않는다
4. `_output/` 복사본만 수정한다

## 핵심 규칙
- 매칭: H12 품번 정규화 기준 (SUB품번 매칭 금지)
- 셀 분리: C안 통일 (AB5:AJ9 / AB10:AJ12 / AB13:AJ14)
- 폰트: SP3M3=85pt검정, 공용품번=110pt(LH빨강/RH파랑), RSP=80pt검정
- RSP 부분색상: SC=빨강, SE=초록, PC=보라, HE=주황, AC=갈색
- 배경: AB1:AJ4=하늘색, AB5:AJ14=흰색
- 안쪽 구분선: AB5:AJ14 내부선 없음
- 뷰: Normal View 강제, 페이지 나누기 제거

## 금지사항
- 사용자 확인 없이 기준/방식 변경
- _backup 원본 직접 수정
- SUB품번 기준 매칭
- C안 셀 분리 외 다른 병합 구조 사용
- 검증 없이 전체 적용
