# Fast / Full Lane 판정

영향 반경으로 판정. 파일 수 무관.

## Fast Lane (전부 만족 시)
- 수정 범위 국소적 (한 시트/함수/블록)
- 구조 안 건드림 (시트명/컬럼/경로/참조체계 불변)
- 업무 규칙 불변 (산식/단가/판정 로직)
- 실패 영향 낮음
- → plan.md 생략, 구현→검증→커밋

## Full Lane (하나라도 해당 시)
- 시트 간 참조/수식 체계/스크립트 로직/컬럼 구조 변경
- 업로드 양식/외부 인터페이스 영향
- 실무 파일의 값/구조/양식 영향
- → research.md → plan.md 승인 → 구현 → 검증 → 커밋
