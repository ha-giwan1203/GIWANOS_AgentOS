# 04_생산계획

이 폴더는 생산지시서 원본, 자동화본, VBA 패치 스크립트를 관리하는 도메인 루트다.

## 현재 내용

- 생산지시서 원본/자동화본 `xlsx`, `xlsm`
- VBA 패치 스크립트
  - `create_plan_output.py`
  - `vba_inject.py`
  - `vba_patch.py`

## 운영 포인트

- 원본 파일과 자동화본 버전이 함께 존재한다.
- VBA 패치와 출력 시트 생성은 파일 구조 검증과 같이 관리해야 한다.

## 권장 작업

- 원본 대비 자동화본 차이 기록
- 패치 적용 후 smoke check
- 실행 순서와 결과물 표준화
