# 04_생산계획 도메인 규칙

## 목적

생산지시서 원본과 자동화본, VBA 패치 스크립트, 출력 시트 생성 로직을 함께 관리한다.

## 읽기 순서

1. `README.md`
2. `STATUS.md`
3. 스크립트와 `xlsm` 파일

## 작업 원칙

- `xlsm` 수정 전 원본과 자동화본의 역할을 먼저 구분한다
- VBA 패치 후에는 시트 존재 여부, 출력 시트, 주요 수식 범위를 최소 검증한다
- 실사용 파일명과 날짜 버전은 임의 축약하지 않는다
- 전역 우선순위와 인수인계는 `90_공통기준/업무관리/TASKS.md`, `HANDOFF.md`를 따른다
- ERP/MES 잔존 데이터 회복(D0 중복 청소·MES 잔존 등): `90_공통기준/erp-mes-recovery-protocol.md` 참조
