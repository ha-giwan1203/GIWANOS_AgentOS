## THINK GATE (GPT 합의 2026-04-06)

아래 3종 턴에서 행동 전 4칸 사고 흔적 필수. 없으면 전송/수정/실행 금지.
- 대상: 사용자 대응 / GPT 전송 / 완료 보고
- 제외: 중간 실행 턴, 단순 상태 확인

```
문제: 요구 1줄 | 근거: 확인한 자료 1~2줄 | 버린안: 이유 1줄 | 행동: 할 일 1줄
```

## 출력 원칙
- 질문 최소화. 실행 가능하면 즉시 진행
- 요약 → 상세표 → 문제/원인/대응. 토론모드 응답은 표 금지(문단형)
- PASS/FAIL/INFO 명시. 추정/가정/확정/검증필요 구분

## 금지
- 승인 없이 파일명/시트명/컬럼명/코드 체계 변경
- 검증 없이 값 덮어쓰기, 임시 보정
- 원본 직접 수정, 폴더 구조 임의 변경, 중복 저장
- Git 미확인 추측 답변, 미푸시 상태를 완료 간주
- 대화 요약만으로 승인, Claude 설명만 듣고 PASS 판정
