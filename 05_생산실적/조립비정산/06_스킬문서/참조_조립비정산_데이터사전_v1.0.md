### 3. 조립비_관리_DB__{MM}월.xlsx

수식 기반 워크북. 아래 시트들로 구성:

#### 라인 시트 (SD9A01, ANAAS04, DRAAS11, SP3M3, HASMS02, HCAMS02, WAMAS01, WABAS01, WASAS01, 이너센스ASSY)

Row 0: 상위 헤더 (GERP 실적, GERP 금액, 구 ERP 실적, 구 ERP 금액)
Row 1: 하위 헤더
Row 2+: 데이터
마지막 행: SUM 합계

| Col (1-based) | 헤더 | 수식여부 | 설명 |
|---|---|---|---|
| A (1) | 품번 | 고정 | 기준정보 품번 |
| B (2) | 조립업체코드 | 고정 | '0109' |
| C (3) | 조립라인코드 | 고정 | 라인코드 |
| D (4) | 조립품번 | 고정 | 조립품번 |
| E (5) | Usage | 고정 | 1 또는 2 |
| F (6) | 단가구분 | 고정 | 정단가/미단가 |
| G (7) | 단가 | 고정 | 기준정보 단가 |
| H (8) | 차종 | 고정 | 차종코드 |
| I (9) | GERP 주간실적 | 수식 | =IFERROR(VLOOKUP(A,Q:R,2,0),0) |
| J (10) | GERP 야간실적 | 수식 | (있는 경우) |
| K (11) | GERP 주간금액 | 수식 | =G×I |
| L (12) | GERP 야간금액 | 수식 | =G×J |
| M (13) | 구ERP 주간실적 | 수식 | =IFERROR(VLOOKUP(A,구ERP!B:C,2,0),0) |
| N (14) | 구ERP 야간실적 | 수식 | (있는 경우) |
| O (15) | 구ERP 주간금액 | 수식 | =G×M |
| P (16) | 구ERP 야간금액 | 수식 | =G×N |
| Q (17) | 제품번호 | 데이터 | GERP 피벗 (매월 교체) |
| R (18) | 수량 | 데이터 | GERP 피벗 수량 |

#### GERP 시트
전 업체 GERP 가공 데이터. 라인시트의 참조용.

#### 구ERP 시트
Row 2: 헤더 (기종, 품번, 입고수량)
Row 3+: 품번-수량 피벗 (전 업체 합산)
라인시트의 VLOOKUP(A, 구ERP!B:C, 2, 0) 참조 대상.

#### 정산금액 시트
각 라인시트 합계행을 참조하는 요약. 수식 기반.

> - SP3M3: `gerp_ngt_amt = gerp_ngt_qty × 170` (고정단가)

> - SD9A01 (단가 > 500): `gerp_ngt_amt = gerp_ngt_qty × 기준단가` (기본, 야간수당 없음)
> - SUB 라인: `gerp_ngt_amt = 0` (야간 없음)

## LOT번호 주야간 분류 규칙

LOT번호 형식: YYMMDD + 구분자 (예: 260202A)
- A: 주간 1조
- B: 야간
- S: 주간 지원
- C: 주간 기타
