# 업무리스트 — AI 업무공유 기준 저장소

자동차 부품 제조업(삼송 G-ERP) 업무 자동화 저장소.
**이 GitHub 저장소는 GPT와 Claude가 동일한 규칙, 상태, 작업 큐, 인수인계 기록을 공유하는 AI 업무공유 기준 저장소이다.**

## AI 세션 시작 순서

새 AI 세션을 시작할 때 아래 순서로 읽고 작업을 이어받는다.

| 순서 | 파일 | 목적 |
|------|------|------|
| 1 | `90_공통기준/업무관리/HANDOFF.md` | 이전 세션 목적·변경·미완료·주의사항 |
| 2 | `90_공통기준/업무관리/STATUS.md` | 전체 운영 현황 및 완료/미완료 항목 |
| 3 | `90_공통기준/업무관리/TASKS.md` | 즉시 작업·대기 작업·우선순위 |
| 4 | `CLAUDE.md` (루트) | 프로젝트 전체 운영 원칙 |
| 5 | 도메인 `CLAUDE.md` | 해당 업무 세부 규칙 |

## 운영 원칙 (3줄 요약)

1. **업무리스트 폴더 = Google Drive 동기화 폴더.** 작업과 원본 기준은 동일 위치이다.
2. **GitHub = AI 업무공유 기준 저장소.** 규칙·상태·인수인계 문서는 GitHub에서 관리하고 AI 세션 간 공유한다.
3. **Notion = 사람용 현황판.** AI 기준 저장소가 아니다. 완료·오류 알림은 Slack, 최종 이력은 GitHub에 남긴다.
